# KaiheilaBot

[WIP] Kaiheila bot 工具人羊驼

## Getting started

```shell
go mod tidy
```

```shell
go build .
```

## API

Backend: 
* [网易云音乐 API](https://github.com/Binaryify/NeteaseCloudMusicApi)

* [API大全](https://www.alapi.cn/)

## USE
### 智能机器人
`羊驼`  +  `你想查询的关键词`

### 历史上的今天
`羊驼` + `历史上的今天`

### 微博热搜
`羊驼` + `微博热搜`
