```

go build .
```

## USE
### 智能机器人
`羊驼`  +  `你想查询的关键词`

目前支持的功能：天气、翻译、藏头诗、笑话、歌词、计算、域名信息/备案/收录查询、IP查询、手机号码归属、人工智能聊天

### 历史上的今天
`羊驼` + `历史上的今天`

### 网络热搜
支持的热搜：知乎热搜，微博热搜，百度热搜，B站日榜、B站综合榜，豆瓣新片、豆瓣新剧，小黑盒新闻

eg：`羊驼` + `微博热搜`
