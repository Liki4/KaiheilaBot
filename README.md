# KaiheilaBot

[WIP] Kaiheila bot 工具人羊驼

## Getting started

```shell
go mod tidy
```

```shell
go build .
```

## API

Backend: [网易云音乐 API](https://github.com/Binaryify/NeteaseCloudMusicApi)

## USE
### 智能机器人
`/ff`  +  `你想查询的关键词`

### 历史上的今天
`/today`
